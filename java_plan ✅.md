## 一、项目结构

**前后端分离架构**：

- **后端**：使用 Spring Boot 提供 RESTful API 接口  
    
- **前端**：使用 Vue 实现网页交互  
    
- **数据库**：使用 MySQL 存储用户、商品、订单等信息  
    
- **ORM**：通过 JPA（Hibernate 实现）管理数据库实体，实现对象化操作  
    
- **本地运行**

---

## 二、开发流程

1. **数据库设计**：确定数据关系，建立数据库  
    
2. **后端API实现**：编写 Spring Boot 控制层、服务层、数据访问层  
    
3. **前端页面开发**：Vue 组件化实现页面交互  
    
4. **前后端联调**：通过 Axios 调用 Spring Boot 接口  
    
5. **本地运行**
